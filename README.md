![screenshot]('./screenshot/v1.gif')

## Included
### react-native + redux + immutable + redux-thunk + normalizr + redux-actions + redux-devtool ...

## How to use
### iOS
```bash
$ npm install -g react-native-cli
$ npm install 
$ react-native run-ios
```

### android
```bash
$ npm install -g react-native-cli
$ npm install 
$ react-native run-android
```
